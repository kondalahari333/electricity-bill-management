#electricity bill calculator
##description
This java program calculates the electricity bill based on the number of units consumed.
it uses real life billing logic with conditional statements
----
##rules of billing
-first 100 units=1 rupee per unit
-Next 100 units(101-200)-2 rupee per  unit
-Above 200 units-3 rupee per unit
 ##surcharge rules
 -if total bill amount is more than 500.
 ->10% surcharge will be added.
 ---
 #concepts used
 -java
 -if-else conditions
 -basic arithmetic operations
 ---
 #how to run
 1.compile the programme
 2.run the programme
